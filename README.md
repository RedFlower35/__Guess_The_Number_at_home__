# 🎮 猜數字大挑戰 | Guess The Number

一個充滿現代感且具備流暢動畫的猜數字遊戲。專為 GitHub Pages 打造，挑戰你的直覺，在 10 次機會內猜中目標數字！

## ✨ 特色
- 🚀 **現代化 UI**: 使用 Tailwind CSS 4.0 打造。
- 🎬 **流暢動畫**: 使用 Framer Motion 實現細膩的互動效果。
- 📱 **響應式設計**: 完美適配手機與桌面端。
- 📊 **紀錄追蹤**: 自動記錄你的猜測歷程並提供提示。

## 🛠️ 本地開發

**前置需求:** Node.js (建議 v18+)

1. 安裝依賴:
   ```bash
   npm install
   ```
2. 啟動開發伺服器:
   ```bash
   npm run dev
   ```

## 🚀 部署至 GitHub Pages

本專案已配置好 Vite 的 `base` 路徑。請依照以下步驟部署：

1. **推送程式碼**: 確保你的程式碼已推送到 GitHub 上的 `main` 分支。
2. **設定 GitHub Actions**:
   - 在 GitHub 儲存庫頁面點擊 **Settings** -> **Pages**。
   - 在 **Build and deployment** > **Source** 選擇 **GitHub Actions**。
3. **完成**: GitHub 會自動偵測並開始部署。稍等幾分鐘後即可透過專屬網址存取！
